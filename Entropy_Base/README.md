# 熵增基座 (elaina-plugin-Entropy_Base)

一个装在 **ElainaBot** 上的「游戏查询插件合集」。装好基座后，在后台一键下载你想要的游戏插件，机器人就能在 QQ 群/私聊里查战绩、查数据、定时推送。

> 插件来自 **熵增开发组**（Entropy-Increase-Team），开箱即用，无需懂代码。

---

## 主打功能：三大游戏查询

| 游戏 | 能做什么 |
|---|---|
| **三角洲行动** | 查战绩 / 段位 / 武器数据，远征仓库、每日特勤处「家菜」行情订阅推送等 |
| **终末地（明日方舟：终末地）** | 查游戏数据、扫码登录绑定、开服/活动信息查询与订阅推送 |
| **洛克王国：世界** | 查角色/宠物数据、活动与商城信息，远古商人「远商」上架订阅推送等 |

> 以上都通过后台「插件库」一键安装。安装后在群里发送对应插件的「帮助」即可看到全部指令。

---

## 怎么用（三步）

1. 在 ElainaBot 框架的 **插件市场** 里搜索 **熵增基座**（elaina-plugin-Entropy_Base），点 **下载安装**。
2. 打开机器人后台 → 自定义页面 → **「熵增基座」**。
3. 在 **「插件库」** 里找到想要的游戏（三角洲 / 终末地 / 洛克王国），点 **下载安装** 即可。安装后自动生效，群里就能用了。

> 想更新插件：在「已安装」里点该插件的 **更新**。想卸载：点 **卸载**（数据保留与否以插件为准）。

---

## 常用设置（后台「基座设置」页）

打开后台「熵增基座」→ **基座设置**，常用开关都在这里，改完即时生效：

- **发送图片用 markdown 发送**：官方机器人发图使用markdown，可关掉它改用 QQ 原生发图。
- **图床桶**：markdown 发图时图片存放处，默认免配置，一般不用动。
- **允许普通群成员订阅**：开 = 群里任何人都能设订阅；关 = 群里只有主人能设（私聊不受限）。
- **主动推送机器人**：多个机器人时，指定用哪个发推送；只有一个机器人可不填。
- **框架模块开关**：一键开启 **Playwright 渲染模块**（出图更快）和 **图床服务模块**（markdown 发图需要它）。建议两个都开。

## 订阅推送说明

很多游戏支持「订阅」后定时把行情/活动推送到群里（如三角洲家菜、洛克远商）。

- 群里要先开启 QQ 官方的 **「全量消息」** 才能收到群推送；没开时机器人会提示，并引导主人发送 `全量申请 群号` 获取开通链接。
- 取消订阅随时可用，不受全量限制。

---

## 常见问题

- **群里收不到定时推送？** 先确认该群开了官方「全量消息」（发 `全量申请 群号` 开通）。
- **装完插件提示缺少依赖？** 基座会自动安装；若失败，按日志里给出的 `pip` 命令手动装一次再重载。
